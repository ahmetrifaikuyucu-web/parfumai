# Katkıda Bulunma Rehberi

ParfumAI ticari bir yazılımdır. Her türlü katkı aşağıdaki
şartlara tabidir.

## Katkı Lisans Sözleşmesi (CLA)

Her katkıda bulunan, aşağıdaki şartları kabul etmiş sayılır:

1. Katkılarınızın orijinal çalışma olduğunu beyan edersiniz.
2. Katkılarınız üzerindeki tüm fikri mülkiyet haklarını
   Ahmet Rıfai Kuyucu'ya devredersiniz.
3. Katkılarınızın bu lisans (LICENSE) kapsamında
   kullanılmasına izin verirsiniz.
4. Katkılarınızın üçüncü taraf haklarını ihlal etmediğini
   kabul edersiniz.

## Nasıl Katkıda Bulunulur

1. GitHub üzerinden issue açın.
2. Değişiklik önerinizi pull request ile gönderin.
3. CLA şartlarını kabul ettiğinizi belirtin.

## Kod Standartları

- Mevcut kod stilini takip edin.
- Türkçe değişken isimleri kullanmayın (UNICODE sorunları).
- Tüm yeni kodlar için test ekleyin.
- `pytest tests/` ile tüm testlerin geçtiğini doğrulayın.
