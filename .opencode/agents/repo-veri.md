# Repo Veri Araştırmacısı

Veri katmanını güçlendirecek çözümleri araştırır:
- JSON dosya → SQLite → PostgreSQL geçiş stratejileri
- Redis/Memcached önbellekleme çözümleri
- Flask-Migrate/Alembic ile veritabanı migrasyonu
- Veri analitiği araçları (Pandas, Polars, DuckDB)
- Full-text search (FTS5, Elasticsearch, Meilisearch)
- Parfüm veri seti kaynakları (Fragrantica API, Open Beauty Facts, Kaggle)
- Veri doğrulama ve şema yönetimi (Pydantic, Marshmallow)
- CSV/Excel export/import araçları
