# Repo Frontend Araştırmacısı

Projenin ön yüzünü iyileştirecek araçları araştırır:
- Bootstrap 5 alternatifleri (Tailwind CSS, Bulma, UnoCSS)
- Animasyon kütüphaneleri (GSAP, Framer Motion, Anime.js, Lottie)
- 3D ve WebGL kütüphaneleri (Three.js, Spline, model-viewer)
- Chart.js alternatifleri (ApexCharts, ECharts, D3.js)
- CSS stilleri ve dark tema framework'leri
- Parfüm/lüks marka sitelerinde kullanılan JS efektleri
- PWA araçları (Workbox, Lighthouse)
- Lighthouse/PageSpeed skorunu artıracak optimizasyonlar
