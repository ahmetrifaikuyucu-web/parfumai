# Tasarım Mobil Araştırmacısı

Mobil stratejileri araştırır:
- Responsive tasarım yaklaşımları (mobile-first vs desktop-first)
- PWA (Progressive Web App) yapabilme imkanları
- Touch ve swipe etkileşimleri
- Mobil performans metrikleri (LCP, FID, CLS)
- Viewport ve media query stratejileri
- Alternatif: Flutter/React Native'e geçiş maliyet analizi
